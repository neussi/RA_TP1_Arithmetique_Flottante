\section{Introduction}

Ce rapport presente le travail realise pour le module \textbf{IEEE 754 \& Arithmetique Flottante}
dans le cadre du cours \textit{Realite Augmentee FROM SCRATCH}.

L'objectif est de comprendre et implementer les fondations numeriques indispensables au pipeline
de realite augmentee : representation binaire des flottants, detection des cas speciaux (NaN, infini,
zero negatif), sommation compensee de Kahan, variance de Welford et calcul de l'epsilon machine.

\textbf{Contrainte absolue du cours :} aucune bibliotheque tierce. Uniquement C++17 et STL.

\section{Concepts Theoriques}

\subsection{Representation IEEE 754}

Un flottant \texttt{float} 32 bits est structure en trois champs :
\begin{itemize}
    \item \textbf{Bit 31} : Signe (0 = positif, 1 = negatif)
    \item \textbf{Bits 30--23} : Exposant biaise (8 bits, biais = 127)
    \item \textbf{Bits 22--0} : Mantisse (23 bits implicites)
\end{itemize}

\[
    \text{Valeur} = (-1)^{\text{signe}} \times 2^{\text{exposant}-127} \times 1.\text{mantisse}
\]

L'acces aux bits en C++17 sans violer le \textit{strict aliasing} :
\begin{lstlisting}[language=C++, caption=Lecture des bits IEEE 754]
uint32_t bits;
std::memcpy(&bits, &x, sizeof(float));
int sign = (bits >> 31) & 1;
int exp  = (bits >> 23) & 0xFF;
int mant = bits & 0x7FFFFF;
\end{lstlisting}

\subsection{Sommation de Kahan}

La sommation naive accumule des erreurs d'arrondi. L'algorithme de Kahan maintient
un compensateur \texttt{c} pour recuperer les bits perdus :

\begin{lstlisting}[language=C++, caption=Algorithme de Kahan]
float kahanSum(const std::vector<float>& v) {
    float sum = 0.0f, c = 0.0f;
    for (float x : v) {
        float y = x - c;
        float t = sum + y;
        c = (t - sum) - y;
        sum = t;
    }
    return sum;
}
\end{lstlisting}

\subsection{Variance de Welford}

Algorithme numeriquement stable pour le calcul de la variance :

\begin{lstlisting}[language=C++, caption=Algorithme de Welford]
float varianceWelford(const std::vector<float>& v) {
    float mean = 0.0f, M2 = 0.0f;
    int n = 0;
    for (float x : v) {
        ++n;
        float delta = x - mean;
        mean += delta / n;
        M2 += delta * (x - mean);
    }
    return (n < 2) ? 0.0f : M2 / (n - 1);
}
\end{lstlisting}
